# Simple cross-platform client application to test Go gRPC service 
https://github.com/zs-dima/auth-service

Client provides UI for basic operations: login, view users, edit users.

## Features
- [x] JWT authentication
- [x] User management
- [x] User settings
- [x] Adaptive theme
- [x] Localization
- [ ] Local storage
- [ ] Event Bus notifications
- [ ] Test coverage
- [ ] Documentation

Welcome to contribute, suggest and discuss improvements.
